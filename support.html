Common questions

My streak disappeared

Streaks reset if you miss a day. If you have streak freeze enabled, the app uses one charge automatically when you miss. Charges can be earned via in-app currency ("traces") in the Shop.

I bought "Remove ads" but still see ads

Open Settings → Purchases → Restore purchase. This re-checks your Apple ID for the purchase and activates ad-free mode. Same flow works for the Pro Bundle and Daily 2×.

The leaderboard isn't updating

Leaderboards require iCloud. Check Settings → Apple ID → iCloud → iCloud Drive is enabled. Recent results sync within a minute under normal connectivity.

How are puzzles generated?

Every player solves the same five puzzles each day. Puzzles are deterministically generated on-device from a daily seed (UTC date + cryptographic hash). No server is involved.

I want to delete my data

Deleting the app removes all local data. To remove your anonymous leaderboard records from iCloud, send an email with the request and your anonymous player ID (visible in the app's Settings).
